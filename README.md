# obselate/tap

Homebrew tap for [obselate](https://github.com/obselate) tools.

## Install

```bash
brew install obselate/tap/bntui
```

## Formulas

| Formula | Description |
|---------|-------------|
| [bntui](https://github.com/obselate/bntui) | Terminal block explorer for Blocknet privacy blockchain |
